Codebase style is Java 17 with Chinese Javadocs/comments, fluent setters returning self(), and CRTP-style BaseUIElement<T extends BaseUIElement<T>> inheritance. UI components commonly extend BaseUIElement and override drawSelf plus selective event hooks. Children are stored sorted by z ascending, then traversed forward for render and reverse for hit-testing. Rendering code relies on GuiGraphics/PoseStack conventions and avoids unnecessary per-frame allocations where possible.
